# Cursor エディタ ショートカットキー一覧

## 基本操作

| 操作 | ショートカットキー |
| ---- | ----------------- |
| AIチャットウィンドウを開く | `Ctrl + Shift + C` |
| ファイル新規作成 | `Ctrl + N` |
| ファイルを保存 | `Ctrl + S` |
| すべてのファイルを保存 | `Ctrl + Alt + S` |
| ファイルエクスプローラーの表示/非表示 | `Ctrl + B` |
| ターミナルの表示/非表示 | `Ctrl + `` (バッククォート) |
| 設定を開く | `Ctrl + ,` |
| コマンドパレットを開く | `Ctrl + Shift + P` |

## 実践的な使い方のコツ

### ファイル操作の効率化

1. **クイックオープン（Ctrl + P）の活用**
   - ファイル名の一部を入力するだけで素早くファイルを開ける
   - 例：「basic」と入力すれば `basic_template.html` が見つかる
   - プロジェクトが大きくなっても効率的にファイルを見つけられる

2. **ファイル保存の基本**
   - `Ctrl + S`：現在のファイルを保存
   - `Ctrl + Shift + S`：名前を付けて保存（別名で保存）
   - 新しいファイルを作成したら必ず適切な拡張子で保存する

### AIチャット機能の効果的な使い方

1. **コード選択 + AIチャット**
   - コードを選択してから `Ctrl + Shift + C` でチャット
   - 選択したコードについて具体的に質問できる
   - 「このコードの意味は？」「エラーの原因は？」など

2. **効果的な質問の仕方**
   - 具体的で明確な質問をする
   - 「このJavaScriptコードの5行目でTypeErrorが発生する原因は？」
   - 曖昧な質問よりも詳細な質問の方が的確な回答を得られる

### マルチカーソル機能の活用

1. **基本的な使い方**
   - `Alt + クリック`で複数の場所にカーソルを設置
   - 同じ編集を複数箇所で同時に実行できる

2. **実用例**
   - 複数の `<p>` タグを一度に `<div>` に変更
   - 複数の変数名を一括で変更
   - リストの項目を一度に編集

### ファイル管理のベストプラクティス

1. **適切なファイル名**
   - 英数字とハイフン、アンダースコアを使用
   - 日本語や特殊文字は避ける
   - 例：`my_profile.html`, `about-us.html`

2. **フォルダ構成**
   ```
   📁 project/
     ├── index.html
     ├── 📁 pages/
     │   ├── about.html
     │   └── contact.html
     ├── 📁 css/
     │   └── style.css
     └── 📁 images/
         └── logo.png
   ```

### よくあるトラブルと解決法

1. **ファイルが正しく表示されない**
   - 拡張子が `.html` になっているか確認
   - ファイル名に日本語や特殊文字が含まれていないか確認

2. **コードが文字として表示される**
   - HTMLタグが半角文字で書かれているか確認
   - ファイルの文字エンコーディングが UTF-8 になっているか確認

3. **保存したファイルが見つからない**
   - 保存場所を確認（どのフォルダに保存したか）
   - ファイルエクスプローラー（Ctrl + B）で確認

### 学習効率を上げるコツ

1. **ショートカットキーを段階的に覚える**
   - まずは基本的なもの（Ctrl + S, Ctrl + P, Ctrl + Shift + C）から
   - 慣れてきたら編集系のショートカットを追加

2. **開発者ツールとの連携**
   - F12で開発者ツールを開く
   - ブラウザで確認しながらCursorで編集
   - リアルタイムでコードの動作を確認

3. **継続的な学習**
   - 毎日少しずつでもコードを書く
   - 分からないことはAIチャットで質問
   - 作った作品を保存して成長を実感する
